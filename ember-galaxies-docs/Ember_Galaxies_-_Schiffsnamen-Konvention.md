# Ember Galaxies - Schiffsnamen-Konvention

## Design-Philosophie der Namen
- **Kleine Schiffe**: Insekten-Namen (schwarmfähig, zahlreich)
- **Mittlere Schiffe**: Klassische Militär-Begriffe (etabliert, vertraut)
- **Große Schiffe**: Epische Klassen-Namen (Macht und Prestige)

## Komplette Schiffsliste mit Antriebsanforderungen

### **Klasse 1: Scouts (System-intern, Basis-Antrieb)**
```
1. Fly           → schnell, winzig, nervt große Gegner
2. Bumblebee     → ausgeglichen, "summt" herum, defensive Taktiken
```

**Abkürzungen:** FL, BB

### **Klasse 2: Corvettes (System-intern, verbesserter Antrieb)**
```
3. Corvette      → erste "echte" Kampfschiffe
```

**Abkürzungen:** COR

### **Klasse 3: Cruisers (Intersystem, neuer Antrieb erforderlich)**
```
4. Light Cruiser   → erster intersystemarer Antrieb erforderlich
5. Heavy Cruiser   → verstärkte Variante, kann übersprungen werden
```

**Abkürzungen:** LC, HC

### **Klasse 4: Capital Ships (Galaktisch, wieder neuer Antrieb erforderlich)**
```
6. Battleship           → direkter Sprung möglich (LC/HC überspringen)
```

**Abkürzungen:** BS

### **Klasse 5: Super Capital Ships (Endgame-Antriebe)**
```
7. Battleship of Nexus Class    → vorletztes Flaggschiff
8. Battleship of Ember Class    → DAS ultimative Schiff des Spiels!
```

**Abkürzungen:** BS-N, BS-E

### **Klasse 6: Support Titans (spezielle Funktion)**
```
9. Carrier Titan                → Hangar-Monster für Mega-Flotten-Transport
```

**Abkürzung:** CT

## Vollständige Liste mit Antriebssystem-Integration:

```
1. Fly                          (FL)  ← System-intern, Basis-Antrieb
2. Bumblebee                    (BB)  ← System-intern, Basis-Antrieb
3. Corvette                     (COR) ← System-intern, besserer Antrieb
4. Light Cruiser                (LC)  ← Intersystem, neuer Antrieb erforderlich!
5. Heavy Cruiser                (HC)  ← Intersystem, kann übersprungen werden
6. Battleship                   (BS)  ← Galaktisch, wieder neuer Antrieb!
7. Battleship of Nexus Class    (BS-N) ← Nexus-Antrieb
8. Battleship of Ember Class    (BS-E) ← Ultimate! Ember-Antrieb
9. Carrier Titan                (CT)  ← Spezial-Support, Mega-Hangar
```

## Strategische Entwicklungspfade:

### **Standard-Progression:**
Fly → Bumblebee → Corvette → Light Cruiser → Heavy Cruiser → Battleship → Nexus Class → Ember Class → Carrier Titan

### **Sprung-Strategien (wie in WoG):**
- **Heavy Cruiser überspringen**: Light Cruiser → direkt zu Battleship (galaktischen Antrieb erforschen)
- **Früher Nexus-Focus**: Battleship → direkt zu Nexus Class
- **Support-Spezialisierung**: Früh zu Carrier Titan für Mega-Logistik

## Charakteristika der Namenskonvention:

### **Fly & Bumblebee:**
- Thematisch: Kleine, schwarmfähige Einheiten
- Charakteristisch: Fly = schnell/aggressiv, Bumblebee = defensive/ausgewogen
- Memorable: Jeder kann sich Insektenschwärme vorstellen

### **Corvette bis Battleship:**
- Vertraut: Etablierte militärische Terminologie
- Progression: Logische Größen-/Machtsteigerung
- International: Funktioniert auf Deutsch und Englisch

### **Nexus & Ember Class:**
- **Nexus**: Impliziert End-Game Power, Verbindungspunkt aller Technologie
- **Ember**: Perfekte Verbindung zum Spielnamen und -thema - DAS ultimative Schiff
- **Eigenständig**: Komplett eigene Namen, keine WoG-Ähnlichkeit

### **Carrier Titan:**
- Support-Funktion: Wie WoG-Dino, aber eigener Name
- Titan-Bezeichnung: Impliziert gewaltige Größe und Hangar-Kapazität
